== Description ==
After installation new post type register "Authors" you can add authors personal detail for example:
- First name
- Last name
- Biography
- Facebook URL
- Linkedin URL
- You can link that author to wordpress users and on frontend this users post will list at the end
- Authorâ€™s image
- Image gallery

You can accessed that author post by his first and last name should be like this 
http://example.com/authors/first_name_field-last_name_field

== Installation ==
1. Upload "wp-authors-profile" to the "/wp-content/plugins/" directory.
2. Activate the plugin through the "Plugins" menu in WordPress.
